## Recommended Crop Strategy

### `05-results-panel.png`
- Scene 1: crop tight on the highest-signal score card or preview corner.
- Scene 2: widen to include tabs, findings, and preview.
- Scene 7: use the broadest readable results crop.

### `03-scoring-prompt-overview.png`
- Scene 3: begin centered, then drift left so source and inputs become clearer.
- Scene 8: use the fullest, widest version for the system reveal.

### `07-source-panel.png`
- Keep the selected Huntr job and populated description visible in the same frame.
- Favor readability over style here. This is the "real workflow" proof point.

### `09-appearance-overview.png` and `10-appearance-panel.png`
- Start on `10-appearance-panel.png` for control.
- Transition to `09-appearance-overview.png` to connect control to preview.

### `11-export-actions-resume.png` and `12-export-actions-cover.png`
- If you want a cleaner ending, use resume export only.
- If you want to reinforce the paired-output story, use a short cut from resume export to cover export.
